# kindergarte
基于ssm的育儿园信息管理系统

开发环境：

（1）操作系统：Windows 10

（2）电脑硬件要求：4G内存或者更高

（3）数据库：mariadb-10.3.9-winx64

（4）Web服务器：apache-tomcat-8.5.31

（5）浏览器：Google Chrome
